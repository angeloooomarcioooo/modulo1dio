O que são tags?

Voltar

As tags são etiquetas que não aparecem na tela do computador, mas que servem para orientar o usuario. O texto publicado o que esta na tela do computador foi encaminhado por um usuário do site por meio do canal colaborativo

Conhecendo algumas tags basicas.

Voltar

As tags h1 á h6 são tags usadas para cabeçalhos e elas são usadas para estruturar e organizar o conteúdo em páginas da web.

À TAG P serve para criar um parágrafo

Ja á TAG UL cria um lista não ordenada onde aparecerar antes das informações marcadores. E nessa tag para aparecer os marcadores, deve se usa a TAG LI apos ter digitado a TAG principal de lista ( UL )

Á TAG OL cria uma lista ordenadas de forma característica que são usados em sumários etc.. E pode ser de qualquer tipo, como numerais,letras,algarismo romanos, ou simples símbolos. E nessa lista de itens para aparecer os números ou letras ou algarismo romanos deve usar a TAG LI apos a TAG principal ( ol ) para aparecer.
